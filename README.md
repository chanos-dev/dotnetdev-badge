### ⭐ badge 사용:

#### ✔ small badge:

##### v1 
```
https://profile.dotnetdev-badge.kr/api/v1/badge/small?id={id}&theme={Light,Dark,Dotnet}
```
> parameter - query string

| parameter |  default  | description                                                                |
| :------ | :-------: | :------------------------------------------------------------------------- |
| `id` | - | 닷넷데브 포럼 `사용자 이름` |
| `theme` | `Light` | 뱃지 테마 (`Light`, `Dark`, `Dotnet`) |
